Seja Muito Bem Vindo!

	Essa página será o guia para o restante do site, que pode ser acessado no 
menu ao topo.

Qual a Função Desse Site?

	A finalidade desse site é o aprendizado, por tanto nem todo o conteúdo 
contido aqui terá um contexto, algumas coisas podem ter utilidade, enquanto 
outras podem ser usadas apenas para teste e ou demonstração.

Currículo

	A página currículo tem a função principal de demonstrar a utilização de 
tabelas em uma página, porém nela também podemos ver as informações 
profissionais do desenvolvedor desse site. Observe que são demonstradas 
algumas tabelas diferentes, justamente para exemplificar que uma tabela pode 
ser montada com quantas linhas ou colunas forem necessárias.

Contato

	Em contato a principal funcionalidade é demonstrar a utilização de imagem 
junto com texto, lembrando que essas imagens também podem ser utilizadas 
para exibir símbolos ou qualquer outra coisa que for necessária. É bom notar que 
essas imagens foram pegas de fontes diferentes, porém foram configuradas para 
parecerem estar dentro do mesmo contexto. Além disso nessa página também 
temos as informações do desenvolvedor do site, que caso seja exibido para 
público um dia e algum usuário tiver interesse, pode entrar em contato para
negociar.

Comentários

	Nessa parte é exibido um sistema de comentários, onde tem um campo 
para identificar o remetente e um campo da mensagem em si, apesar de ser um 
sistema simples, nele é utilizado um banco de dados mysql. O objetivo seria 
enviar mensagens para o proprietário do site de forma simples, sem precisar de 
outro meio de comunicação, sendo que essas mensagens ficam visíveis para os 
outros visitantes do site, permetindo assim que eles se comuniquem entre eles.

Vídeos

	Aqui é demonstrada a exibição de vídeos, nesse caso foi utilizada uma api 
do youtube, que é uma plataforma excelente, de fácil acesso, simples e com um 
desempenho excelente. Também seria possível colocar os vídeos no site sem 
utilizar o youtube, porém na forma em que ele está, fica mais leve, pois não 
precisa hospedar os vídeos, deixando assim a navegação mais rápida, fora que 
o youtube é uma plataforma tão popular que é capaz de deixar os usuários mais 
confortáveis do que se o vídeo fosse hospedado no site, tornando essa decisão 
mais simples e mais eficiente. É bom notar que os vídeos foram organizados de 
formas diferentes, para demonstrar que é possível exibidos separados ou juntos, 
ou em tamanhos diferentes.
